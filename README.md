# Bookfinder

A modern book discovery platform built with Next.js and TypeScript. Features intelligent search, advanced filtering by genre and availability, personalized recommendations, and a streamlined reading experience—helping readers find their next great book quickly.

## Tech Stack

- **Next.js 14** · **TypeScript** · **Tailwind CSS** · **Radix UI**

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Install & run

```bash
# Install dependencies
npm install

# Start development server (http://localhost:3000)
npm run dev

# Build for production
npm run build

# Run production build
npm start

# Lint code
npm run lint
```
